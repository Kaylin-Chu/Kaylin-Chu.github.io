Control flow and Loops

Loops are a sequence of instructions in code that repeats as many times as the user sets, usually until a condition is met. Control flow is the order that coded instructions happen. Code is run top line to bottom line unless the control flow is changed with code. Conditionals can skip code or loops can loop a section of code over and over changing the control flow. This is an example process of cooking Japanese curry for dinner. Gather the ingredients (grocery store, pantry, fridge etc) Prepare the ingredients (cut chicken, carrots, potatoes, onions) Cook the ingredients (saute, stir fry) Add water & curry mix Simmer until ready The instructions to cook dinner will be followed top to bottom, just like how a computer reads code, this is Control Flow. In loops, these steps will only be done if conditions are met. So 'cooking dinner' conditions would be the user is hungry, it's the evening, no other plans (leftovers, dinner out). For example, this loop would be run up to 7 times a week.

What is the DOM?

DOM stands for Document Object Model DOM is a programming interface for web documents. The browser creates DOM when loading HTML documents and arranges the elements in tree-like structure. The DOM represents the webpage so the user (via programs) can interact with document structure, style, and content. JavaScript is one of the programs the user can use to interact with content within a website. The DOM is represented as a Javascript Object and is the exact same as the HTML file. HTML is the language of the web and has specific syntax and rules - not general grammar. The browser turns HTML tags into elements that form the trees, the browser knows how to because of the DOM. ← The aforementioned tree-like structure

I have also seen the DOM described as "A Web API used to build websites, built using multiple APIs that work together"
API = Application Programming Interface

Explain the difference between accessing data from arrays and objects.

To access data from arrays, the user must refer to the element number, the number at which the element is placed. In coding, counting starts at 0, similar to how buildings start with a ground level then the first floor and so on. To access data from objects, the user can use dot notation or bracket notation. Nested objects can be output(ted) too, nested objects refer to objects within objects. Arrays and objects often get compared because they both data structures to hold information. An easy way to tell the difference is arrays are usually in square brackets [array] and objects are usually curly brackets {object}

Explain what functions are and why they are helpful.

Functions are blocks of code that complete a task or tasks. Functions aid in breaking a large program into smaller, more manageable pieces (functions). Functions are easily read and once defined, can be reused multiple times without having to be written out multiple times. Because functions are easily read and understandable, it is easy to fix mistakes/remove bugs.
